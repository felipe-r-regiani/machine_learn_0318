# machine_learn_0318

Este repositório contém um projeto de **Regressão Linear Simples** aplicado à predição de preços de imóveis, utilizando o dataset real de King County (EUA). O objetivo é entender a relação estatística entre a área construída e o valor de mercado das propriedades.

## 📂 Conteúdo

* **kc_house_data.csv**: Conjunto de dados contendo informações detalhadas sobre vendas de casas em King County, incluindo preços, metros quadrados, número de quartos e localização.
* **regres_kc.ipynb**: Notebook que realiza a análise completa, desde o carregamento dos dados com Pandas até a implementação da regressão linear. Inclui o cálculo do coeficiente de correlação (r), do coeficiente de determinação ($R^2$) e a plotagem da reta de regressão sobre os dados reais com Matplotlib.

## 🛠️ Tecnologias e Bibliotecas

As ferramentas centrais utilizadas neste projeto são:

* **Python 3**: Linguagem de programação principal.
* **Pandas**: Manipulação do dataset e limpeza de dados.
* **NumPy**: Suporte para cálculos matemáticos.
* **Matplotlib**: Geração de gráficos de dispersão e visualização da reta de tendência.
* **Jupyter Notebook**: Ambiente interativo para desenvolvimento e visualização.
